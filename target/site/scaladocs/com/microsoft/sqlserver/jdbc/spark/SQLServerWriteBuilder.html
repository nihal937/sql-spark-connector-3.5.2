SQLServerWriteBuilder

class SQLServerWriteBuilder extends WriteBuilder with Logging

SQLServerWriteBuilder implements the WriteBuilder interface for SQL Server bulk write operations. It builds a BatchWrite instance that handles the actual write execution. Automatically detects SaveMode (Overwrite, Append, Ignore, ErrorIfExists) for truncation handling.
